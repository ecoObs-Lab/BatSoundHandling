# BatSoundHandling

Tools for working with bat call recordings. Loading and saving sound files, accessing the sound data, creating sonagrams and spectral data or using a call finder to find calls in sound. These tools are a derivate of the developments used by ecoObs for some of the leading bat call analysis softwares.
